# **App Name**: VerdantLearn

## Core Features:

- Set Creation: Allow users to create and name new study sets.
- Flashcard Addition: Provide an interface to add flashcards to study sets, including fields for the front and back content of each card.
- Study Mode: Implement a study mode where users can review flashcards in a set and flip them to see the answer.
- Progress Tracking: Enable users to mark cards as 'learned' during study mode.
- Mastery Progress Display: Display progress for each set in the landing page, with details of total cards in set vs cards in set that have been mastered.
- Smart Suggestions: AI Tool: Analyze flashcard content and suggest related content or memory aids.

## Style Guidelines:

- Primary color: A refreshing, somewhat saturated green (#74B72E) to evoke learning and growth.
- Background color: Very light green (#F5FAF0), nearly white but with a subtle hint of the primary hue, to ensure comfortable contrast.
- Accent color: A slightly darker analogous green (#5E9725) to highlight interactive elements and key information.
- Font pairing: 'Space Grotesk' (sans-serif) for headings, 'Inter' (sans-serif) for body text, for a modern yet readable look.
- Use clean, line-based icons to represent different subjects or categories, in the primary green color.
- Maintain a clean, card-based layout for study sets and flashcards, emphasizing readability and ease of navigation.
